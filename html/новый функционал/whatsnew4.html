2.  Новая логика заведения контрагентов

При автоматическом добавлении новых контрагентов к списку контрагентов теперь осуществляется проверка не только на совпадение пары ИНН+КПП, но и наименования этого контрагента. Это, в частности, решает проблему отправки платежей в адрес физических лиц с указанием ИНН и КПП банка получателя.

3.  Отображение операций по овердрафтному счету

Добавлено корректное отображение операций по овердрафтному счету, в частности:

  • операции предоставления овердрафтного кредита,
  • операции погашения овердрафта,
  • операции погашения начисленных процентов по овердрафту,
  • операции по гашению просроченной задолженности по овердрафту,
  • операции по гашению просроченных процентов по овердрафту,
  • операции по гашению пени по процентам по овердрафту,
  • операции по гашению пени по кредиту
