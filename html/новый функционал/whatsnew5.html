2.  Корректное отображение разных видов транзакций

Были изменены отображения некоторых видов транзакций в списке счетов и операции, а также в самом слое просмотра транзакций, а именно:
- перевод между счетами
- покупка/ продажа валюты
- кассовые операции
- оплата банковской картой
- возврат денежных средств
- снятие наличных
- внесение наличных

3.  Удаление блока «Устав компании»

Так же убрали блок «Устав компании» из слоя «Открытие счета» на втором шаге регистрации

  • операции предоставления овердрафтного кредита,
  • операции погашения овердрафта,
  • операции погашения начисленных процентов по овердрафту,
  • операции по гашению просроченной задолженности по овердрафту,
  • операции по гашению просроченных процентов по овердрафту,
  • операции по гашению пени по процентам по овердрафту,
  • операции по гашению пени по кредиту
